# Algorithm Study
### **스터디 진행**
```
● 매주 알고리즘 이론 학습과 문제 풀이를 진행
● 시간: 매주 수/토
● 장소: 종로 스터디룸
● baekjoon, SWEA 내 알고리즘으로 진행
● 과제 코드리뷰 -> 알고리즘 문제 풀기 -> 코드리뷰 순으로 진행
```

### **과제**
```
● 다음 스터디 때까지 과제 2개씩
● 백준: B{백준문제번호}_{문제제목}_{이름 이니셜)}, SWEA: S{SWEA문제번호}_{문제제목}_{이름 이니셜)} 형태로 올리기
● 서로의 소스코드에 피드백 할 것 있으면 남기기
```
